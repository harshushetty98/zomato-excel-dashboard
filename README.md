# zomato-excel-dashboard
Interactive Excel dashboard analyzing Zomato sales data using Pivot Tables, charts, and slicers. Provides insights into sales trends, customer ratings, regional performance, and top-performing cities.

📊 Project Description
This project presents an interactive Excel dashboard built to analyze Zomato sales data. The dashboard provides a comprehensive view of business performance through key metrics, visualizations, and filters.

It includes KPI tracking for total sales, average rating, order volume, and average order value. The dashboard enables users to explore trends across months, days, and weeks, along with insights into category-wise and region-wise performance.

Interactive slicers allow dynamic filtering by month, category, and restaurant, making it easy to drill down into specific segments. The project demonstrates the use of Excel for business intelligence and data-driven decision-making.


🧠 Conclusion
The dashboard highlights clear patterns in sales performance, customer behavior, and regional trends. It shows that certain cities and categories consistently contribute to higher revenue, while sales trends vary across time periods.

By combining KPIs, trend analysis, and interactive filters, the dashboard enables quick and effective decision-making. This project demonstrates how Excel can be used as a powerful tool for transforming raw data into meaningful business insights.
