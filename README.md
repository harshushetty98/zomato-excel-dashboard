# zomato-excel-dashboard
Interactive Excel dashboard analyzing Zomato sales data using Pivot Tables, charts, and slicers. Provides insights into sales trends, customer ratings, regional performance, and top-performing cities.


📊 Project Description
This project focuses on analyzing Zomato sales data using Microsoft Excel to generate meaningful business insights. The dashboard was built to track key performance indicators such as total sales, average rating, average order value, total orders, and customer feedback.

The analysis includes multiple visualizations such as monthly and daily sales trends, category-wise comparison (veg vs non-veg), state-wise sales distribution using map visualization, and quarterly performance summaries. Additionally, the dashboard highlights top-performing cities and weekly sales patterns to identify peak business periods.

Interactive filters and slicers were implemented to allow dynamic data exploration, enabling users to drill down into specific segments and uncover trends effectively. The project demonstrates the use of Excel as a powerful data analysis and business intelligence tool.


🧠 Conclusion
The dashboard provides clear insights into sales performance, customer behavior, and regional trends. It reveals that certain cities and food categories contribute significantly to overall revenue, while sales patterns fluctuate across different time periods.

The analysis of monthly, weekly, and daily trends helps identify peak demand periods, enabling better planning and decision-making. Additionally, the relationship between ratings and sales highlights the importance of customer satisfaction in driving business growth.

Overall, this project demonstrates how raw data can be transformed into actionable insights using Excel, supporting data-driven decision-making and improving business performance.
